# User API
you can try it *[here](http://52.54.71.10:8080/swagger-ui.html)*

## Installation Guide

### Prerequisites

- Java 11
- Gradle 7.4
- Git

### Step by step

1. Download sources
```sh
git clone https://github.com/cnorellanan/demo-user.git
```
2. Start project
```sh
cd demo-user
./gradlew bootRun
```

### Try it out!

Access: http://[YOUR_HOST]:8080/demo-user/swagger-ui/
